Crystalis Music Randomizer

Basic usage:

  1. Select the base ROM to use. This may be either the original game ROM or a ROM generated by the Crystalis Randomizer.
  2. Selected a random number to use as the musical seed. It need not match the seed used for the Crystalis Randomizer.
  3. Select options
  4. Click 'Generate ROM'
